Overview

Quick Start

クイックスタート（3ステップ）

1

APIキーを入力

Google（AIza...）または OpenAI（sk-...）

2

台本を入力して [---STEP---] で区切る

各ステップの1行目 = タイトル、残り = 台本

3

「収録スライド生成」ボタンを押す

あとは待つだけ。連番PNGが自動生成されます

Script Format

台本の書き方

はじめに：今日のテーマ ← 1行目がタイトルになる 今日はAIを使った動画制作の 全体像をお話しします。 ← 2行目以降がカンペになる [---STEP---] ← ステップの区切り ステップ1：企画と台本づくり まず最初にやるべきことは、 企画と台本づくりです。 [---STEP---] ステップ2：スライドの準備 次に、スライドを準備します。
Output Order

生成される画像の順番

#種類内容
001ナビ全体の流れ（ステップ1がハイライト）
002箇条書きステップ1 ● 1つ目だけ表示
003箇条書きステップ1 ● 1つ目 + ● 2つ目
004箇条書きステップ1 ● 1つ目 + ● 2つ目 + ● 3つ目
005ナビ全体の流れ（ステップ2がハイライト）
006箇条書きステップ2 ● 1つ目だけ表示
007箇条書きステップ2 ● 1つ目 + ● 2つ目

クリックで送るだけで、箇条書きがポンポン出現します

Download

ダウンロード

ZIP一括ダウンロード

  • 「連番PNG (.zip)」ボタンをクリック
  • ファイル名を自由に設定可能
  • tutorial-001.png 〜 の連番で出力
  • この順番のままOBSで表示すれば収録完了

個別ダウンロード

  • 各スライドにホバー
  • 「保存」ボタンでPNG単体保存
Settings

設定項目

項目説明
LLMプロバイダー箇条書き抽出に使うAI（Gemini / GPT）
LLMモデルGemini 2.5 Flash（高速）/ Pro（高精度）/ GPT-4o
スライドフォントNoto Sans JP / M PLUS Rounded 1c 等
フォントウェイトRegular 〜 Black（太さ）

すべての設定はブラウザに自動保存され、次回以降復元されます

Tips

収録のコツ

  1. ZIPをダウンロードして画像ビューアで開く
  2. OBS / Zoom の画面共有でビューアを映す
  3. 矢印キー（→）でスライドを送りながら話す
  4. カンペを見ながら話す — 暗記不要
  5. 箇条書きがポンポン出るので視聴者のテンポ感も良好
FAQ

よくある質問

Q: 箇条書きが的外れ

→ 各ステップに最低3〜4行の説明文を書いてください

Q: カンペが途切れる

→ 長い台本は自動分割され、各スライドに振り分けられます

Q: APIエラーが出る

→ APIキーの確認 / Gemini APIの有効化 / 無料枠の上限を確認

Q: 停止ボタンを押したら？

→ 生成済みスライドはそのまま残り、ダウンロード可能です
